Sistema de gestión de carnicería

Sistema para uso comercial, incluye módulos tales como:

  • Agregar productos:

    • Permite registrar productos con código, peso, categoría y precio de manera sencilla.

  • Consultar stock:

    • Módulo para verificar el stock actual, controlar mínimos y evitar quiebres.

  • Punto de venta:

    • Sistema de ventas con selección de productos, medios de pago y cálculo automático del total y vuelto a dar.

  • Integración con MercadoPago:

    • Consulta en tiempo real de las últimas 10 transferencias recibidas a travez de una key.

  • Gestión de clientes:

    • Permite registrar clientes, manejar cuentas corrientes y generar pedidos con fecha de entrega.
