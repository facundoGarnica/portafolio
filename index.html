Sistema de gestión de carnicería

Sistema para uso comercial, incluye módulos tales como:

  • Agregar productos:

    • Permite registrar productos con código, peso, categoría y precio de manera sencilla.

  • Consultar stock:

    • Módulo para verificar el stock actual, controlar mínimos y evitar quiebres.

  • Punto de venta:

    • Sistema de ventas con selección de productos, medios de pago y cálculo automático del total y vuelto a dar.

  • Integración con MercadoPago:

    • Consulta en tiempo real de las últimas 10 transferencias recibidas a travez de una key.

  • Gestión de clientes:

    • Permite registrar clientes, manejar cuentas corrientes y generar pedidos con fecha de entrega.

🎮 CodeGames - Plataforma Multi-Juego Indie

Plataforma en desarrollo que permite a desarrolladores y jugadores acceder a múltiples videojuegos indie desde una única cuenta. Actualmente, el único juego disponible es "ArrayRun".

  • Gestión de usuarios unificada:

    Registro y autenticación centralizada para todos los juegos. Un solo inicio de sesión permite acceder a múltiples títulos y guardar el progreso.

  • Soporte multi-juego y sistema de puntuación:

    Permite incorporar nuevos juegos sin reestructurar la base de datos ni el backend. Cada juego mantiene su propia lógica, niveles y puntajes. Se incluye un ranking con los cuatro mejores scores de los jugadores.

  • Selección de niveles y visualización de puntajes:

    Interfaz intuitiva que permite seleccionar niveles y visualizar el puntaje obtenido en cada uno.
